# Chat-app

A chat app created by socket.io, react and redux.


# Usage

- Install
  ```
  npm install
  ```

  or if you use yarn:

  ```
  yarn
  ```
- Run

  ```
  npm start
  ```

  or

  ```
  yarn start
  ```
